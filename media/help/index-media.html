Upload videos

Videos can be uploaded using one of these methods:

  • Drag and Drop - Drag files from the file system to the Media page. The body of the page becomes the drop zone. When the file is dropped, the upload process will begin.
  • File selection dialog - Click  + Add video to open a file selection dialog and then browse and select files to upload.
  • Remote URL - Click  + Upload from URL to specify the location of the source file. Source files must reside on a publicly accessible URL. Click Upload video when finished.

Change page display
  1. Click an icon at the top right of the page to view the videos as cards or as a list.
View video details
  1. Click on a video card or video name to open the Video Details page.
Details
Search videos
  1. Enter search terms and press enter.

The following video fields will be searched:

  • Video ID
  • Name
Filter videos

Filter on Tags

  1. Click the TAGS dropdown.
  2. Enter one or more tags.
  3. Select an operator, Match any or Match all.
  4. Press enter to filter the videos.

Filter on created or modified date

  1. Click the MORE dropdown.
  2. Select Date created or Date modified.
  3. Select a fixed date value or select custom and use the caledat control to select a date or date range.

Filter on custom fields

  1. Click the MORE dropdown.
  2. Select a custom field to filter on.
  3. For a List type custom fields, click on one or more values to filter the videos. If a Text type custom field is selected, enter the appropriate text to filter the videos.
Embed a video in a webpage
  1. In card view, hover over a video card, click the action button (...) and then click Embed on website
  2. In list view, click the action menu (...) next to the video to publish and then click Embed on website
  3. Select a player to use.
  4. Click the preview link to preview in a new browser tab or copy the embed code.

Publish to social media
  1. In card view, hover over a video card, click the action button (...) and then click Publish to Social
  2. In list view, click the action menu (...) next to the video to publish and then click Publish to Social
  3. Click on a destination and click Next .
  4. Enter the text for the post and click Next .
  5. Review the post and click Publish .
