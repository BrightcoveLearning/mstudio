The Quick Edit feature provides a convenient way to edit videos details for one or more videos. Start by selecting one or more videos.

Card view

To select videos, hover over a video and click the checkbox.

List view

To select videos, click the checkbox associated with the video.

When one or more videos is selected, the action bar will appear at the bottom of the page.

Click edit to open the Quick Edit panel.

The Quick Edit panel will display a subset of the video details:

  • Title (if only 1 video is selected)
  • Reference ID (if only 1 video is selected)
  • Description (if only 1 video is selected)
  • Tags to add
  • Tags to remove
  • Related Link URL
  • Related Link Text
  • Custom Fields
Note: For a description of these values, see the Editing Video Details topic.

Use the Quick Edit panel to assign values. When done, click Save to save the changes or Cancel to close the panel.
