The Media page can be used to generate the embed code that is needed to embed a video in a webpage.

Note: The embed code that is generated uses the Brightcove Default Player. This player can be viewed and styled using the Players module in Video Cloud Studio. For information, see Getting Started with the Players Module.

To publish a video to a website, follow these steps.

  1. In the primary navigation, click Media.
  2. To publish a single video to a website, do one of the following:
    • In card view, hover over a video card, click the action button (...) and then click Embed on website
    • In list view, click the action menu (...) next to the video to publish and then click Embed on website
    • Click on a video to open the video details page and click Embed
    Note: Videos that are deactivated cannot be published. For information, see Activating and Deactivating Videos.
  3. (Optional) Click the Preview link to preview the video in another browser tab.
  4. Select the type of publishing code to copy by clicking Standard or Advanced.
    • iFrame - Recommended for most cases. This type places the player inside an <iframe> tag.
    • JavaScript - Allows the player to exist in the the HTML page directly, not in an iframe. This offers developers the benefits of ease of accessing the player and associated properties and events.
    For more information on the types of embed code, see Choosing the Correct Embed Code.
  5. Copy the embed code. When finished, click Done.
